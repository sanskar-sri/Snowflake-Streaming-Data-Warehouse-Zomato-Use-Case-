# Snowflake Zomato Analytics Platform

End-to-end cloud data warehouse built using Snowflake.

## Tech Stack
- Snowflake
- SQL
- Streams & Tasks
- Streamlit

## Features
- Real-time ingestion using Snowpipe
- CDC using Streams
- Dimensional modeling (fact & dimension)
- KPI analytics (orders, revenue)
